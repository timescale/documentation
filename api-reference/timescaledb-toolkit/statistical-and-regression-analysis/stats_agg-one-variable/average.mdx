Calculate a simple average (or mean) from the values in a statistical aggregate.

## Samples

Calculate the average of the integers from 0 to 100:

```sql
SELECT average(stats_agg(data))
  FROM generate_series(0, 100) data;
```

```
average
-----------
50
```
